# react-native-calculator

A simple calculator similar to the iOS one built with React Native.


## Clone Repository using SSH
```
$ git clone git@github.com:drbarzaga/react-native-calculator.git
```

## Install Dependencies
```
$ yarn
```

## Run iOS
```
$ yarn run ios
```

## Run Android
```
$ yarn run android
```
